## Run Locally

Clone the project

```bash
  git clone https://github.com/PushpakRaut/Dice-Game.git
```

Go to the project directory

```bash
  cd Dice-Game
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
